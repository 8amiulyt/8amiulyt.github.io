Daily news in your terminal

2022-07-03

○ The_Daily_Star_CLI

I know it's a huge name :D . OK so it's a script which scrapes the The Daily Star. And serve you the news in your terminal. On that time I was looking for a new idea I was thinking what to do. Then I thought that we need to keep some knowledge about what's happening around us. To keep pace with people around us we need to know what they are up to. Thus I started working on this script. Hope you all will like the script.

○ How does this script work

First of all it uses some tools such as

⎔  curl, sed, grep, echo, paste and perl

Also some tools those are by default not available in distros.

⎔  dmenu, st and w3m

It is a easy process and the script was supposed to be smaller if I would know how to use grep regex. But still somehow it is working xD.

Get The Source Code !!!

○ Installation

$ sudo curl -sL "https://raw.githubusercontent.com/8amiulyt/the-daily-star-cli/main/news" -o /usr/local/bin/news

$ sudo chmod +x /usr/local/bin/news

Execute these two commands by considering the requirements.

Dmenu

st

w3m

These are necessary for the script to be run successfully
